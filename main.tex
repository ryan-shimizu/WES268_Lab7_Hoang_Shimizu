\noindent\textbf{Answer.} 
{
    Based on our results, the (23,12) Golay code performed better than the (24,12) Golay code. At first, this was unexpected because 
    the extra bit increases our minimum distance from 7 to 8 which should improve our error correcting capability. But with a (23,12) setup,
    the codeword fits into one OFDM symbol. With a (24,12) setup, the codeword is spread across two symbols so some symbols will see 0 erasures and 
    some symbols will see 2 erasures. This might cause our overall error to be spread less evenly, leading to a higher decoded BER than our (23,12) setup.
}

\noindent\textbf{Question.}
{ 
    Is the scrambled PAPR always better than the unscrambled? Comment on your results. 
}
\noindent\textbf{Answer.} 
{
    In cases where the input data has repeated structures, then scrambled PAPR generally performs better than unscrambled 
    in terms of PAPR because it "breaks up" the repeated chunks and spreads power more evenly across the spectrum. If the input 
    data is already scrambled, then the scrambler step doesn't necessarily improve the PAPR.   
}

    for each value of DFT (M)?
}
\noindent\textbf{Answer.} 
{
    The localized SC-type waveform has a higher PAPR because the energy is localized to adjacent subcarriers only which leads to greater 
    envelope fluctuation. The interleaved SC-type waveform has a periodic repetition at $\frac{N_{FFT}}{M_{DFT}}$ and as $M_{DFT}$ decreases, 
    the periodicity is more noticeable. The time series looks similar to upsampled single-carrier QPSK.
}

% 3.1.6
\subsubsection{ }

\noindent\textbf{Question.} 
{
    Given what you know about the effect of frequency offset on the constellation of single-carrier
    systems such as ordinary QPSK, can you explain the similarities and/or differences for the
    OFDM case?
}
\noindent\textbf{Answer.} 
{
    For single-carrier systems such as QPSK, a frequency offset would simply mean our constellation has a time-varying phase rotation 
    where all symbols move synchronously. For OFDM, we have multiple subcarriers that could destructively interfere with each other when 
    a frequency offset is applied and breaks the orthogonality. This makes OFDM significantly more sensitive to frequency offsets compared 
    to single-carrier systems. 
}

\noindent\textbf{Question.}
{
    Explain why you think the Golay (24,12) code fails to correct three bit errors when the
    Usable Subcarriers per OFDM symbol is 23. 
}
\noindent\textbf{Answer.} 
{
    The Golay (24,12) can correct up to 3 bit errors at once. But when the number of usable carriers 
    is reduced to 23, the unusable carrier acts as a guaranteed error. When the receiver experiences 
    3 bit errors in addition to the bit error introduced by the bad carrier, it effectively becomes 4 
    errors which exceed the Golay (24,12) correction capabilities.
}

\subsection{Post Lab Question 3d}
\noindent\textbf{Question.} 
{
    Calculate the PAPR (Peak to Average Power Ratio) of an OFDM symbol assuming
    $N=64$ active subcarriers and 8-QAM assuming $Es = 1$. If the transmitted symbols
    are the following:
    $$s(n)=\frac{1}{\sqrt{N}\sum_{k=0}^{N-1}X_ke^{j\frac{2\pi}{N}nk}}$$
    $$n=0,1,...,N-1$$
    $$PAPR=\frac{max(x(n)x^*(n))}{E[x(n)x^*(n)]}$$
}

\noindent\textbf{Answer.} 
{
    We can calculate average power:
    $$E[x(n)x^*(n)]=\frac{1}{N}E[\sum_{k=0}^{N-1}|X_k|^2]$$
    $$=\frac{1}{N}\times N$$
    $$E[x(n)x^*(n)]=1$$
    Peak power occurs when all subcarriers are aligned in phase.
    $$X_k=1$$
    $$x(n)=\frac{1}{\sqrt{N}\sum_{k=0}^{N-1}1}$$
    $$=\frac{N}{\sqrt{N}}$$
    $$=\sqrt{N}$$
    Calculate power:
    $$|x(n)|^2=(\sqrt{N})^2$$
    $$=N$$
    Since we have $N=64$ subcarriers:
    $$\text{max}|x(n)|^2=64$$
    $$PAPR=\frac{64}{1}=64\approx18.06dB$$
}
